# Netflix Content Analysis Dashboard
Overview: This project focuses on analyzing Netflix’s content library using Power BI to uncover trends related to content distribution, genre popularity, and global expansion. The dashboard transforms raw Netflix dataset information into interactive visual insights that help understand how Netflix’s content strategy has evolved over time.

**Tech Stack**: Power BI , DAX (Data Analysis Expressions), Power Query, Excel/CSV Dataset

**Key KPIs**:
Total Content Available,
Total Movies,
Total TV Shows,
Number of Countries Contributing Content,
Ratings Distribution,
Top Genres,
Year-wise Content Growth,

**Business Questions Addressed**
* Which type of content dominates Netflix?
  
* Which countries contribute the most content?

* How has Netflix’s content library grown over time?

* What audience categories are most targeted?

* Which genres are most popular on the platform?

 **Source of Data**: https://www.kaggle.com/datasets/shivamb/netflix-shows
